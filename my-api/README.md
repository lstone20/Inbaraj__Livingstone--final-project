# API to submit contact form, Authenticate Admin user and to create new Admin users

Routes:
1. Contact form submission and retrieval - POST /contact_form/entries ; GET /contact_form/entries
2. Authenticate Admin user - POST /auth
3. Create new Admin users - POST /users

## Instructions
1. Rename `.env.example` to `.env` update any properties as needed

2. `npm start` will start off nodemon API
